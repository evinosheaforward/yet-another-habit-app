# Yet Another Habit App - A minimalist TODO list and habit app

A habit app, for myself - but maybe for you too!

Free to use habit and todo list.

## Features

- Create and track habits
- Create a todo list that combines habits and one-off tasks
- Get detailed views of your habit history
- Create a todo list schedule that adds your habits to your todo list automatically each day
- Create achievements and reward yourself:
  - Complete a habit X times
  - Complete your todo X times
  - Complete all of your habits in a day/week/month
